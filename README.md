# bioinfo_tools
一键完成某项任务的脚本收集

1. kegg富集分析脚本

必备文件为geneid2symbol, hsa00001.keg, diff_gene.txt(可以为symbo或geneid)

放置同一文件夹偶一键运行python kegg.py即可产生kegg_enrichment_result文件夹, 含有csv和png文件
